# Ai4City 新网站

## v0.9 (Nov 27, 2025)  
**全面功能的demo已完成**  

1. 新闻页面现已支持自动生成，点击首页卡片时可以跳转到详情页面，所有功能已经跑通。然而，在前期架构设计时存在一些不足，导致排版和操作比较繁琐。后续更新可以考虑直接嵌入一个Markdown，优化排版操作。  
2. 研究、出版物、资源等部分目前的设想是**直接跳转到外部网页**（例如公众号、DOI链接），不再自己生成内容（**新闻页面是否也考虑如此处理？**）。目前，Research部分的内容框架基本搭建完成，首页可以直接跳转到相关的公众号文章，功能已初步实现，资源和出版物部分内容还需要补充（正在旧网页和公众号里面搜寻）。  
3. 人员模块已基本完成；PI卡片的左侧排版存在一些问题，正在修复中；学生卡片中的个人网页链接已支持点击跳转。  
4. 整体版面在1920x1080分辨率以上的屏幕上显示效果不佳，页面过宽，需要调整适应不同屏幕的布局逻辑。  
5. 仍有一些排版小问题需要修复。

---

**需求与下一步计划**  
1. 可以加入一些实验室文献的DOI或链接，以及公开成果的链接，以丰富网站内容。  
2. 继续完善内容，优化排版效果。  
3. 需要决定是否由网站自行生成详情页面（类似于目前的新闻页面），或是直接链接到外部网页（后者维护更简便）。
