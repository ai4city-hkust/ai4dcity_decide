1. 新闻页面现已支持自动生成，点击首页卡片时可以跳转到详情页面，所有功能已经跑通。然而，在前期架构设计时存在一些不足，导致排版和操作比较繁琐。后续更新可以考虑直接嵌入一个Markdown，优化排版操作。
